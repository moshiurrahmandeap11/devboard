Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM.

  • ByID ==> getElementById()
  • ByClassName ==> getElementsByClassName()
  • ByTagName ==> getElementsByTagName()
  • QuerySelector ==> querySelector()
  • QuerySelectorAll ==> querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

There are few difference between innerHTML, innerText, textContent.

  • innerHTML use to set html element from javaScript ex. getElementById("example").innerHTML = example code;
  • innerText use to get text from other element like h1,h2,p and also it can set a text for a element ex. getElementById("example").innerText
  • textContent is used to get or set the text inside an element, including hidden text. ex. getElementById("example").textContent

Question-3: What is event delegation in the DOM?

Event delegation is a javaScript element which is used to add a single event listener to a parent element instead of multiple event listener to individual child elements.

why we will use event delegation ?

  • Avoid multiple event listener.
  • Only one event listener instead of many.
  • Works Dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where, when an event is triggered on an element, it bubbles up through the DOM tree. In other words, if a click happens on a child element, that event will also be triggered on its parent element.

Question-5: How do you create, add, and remove elements using JavaScript?

In below some example of create , add and remove elements using javaScript.
